Tres

A mobile web development framework

What is Tres?

Tres is a framework for developing JavaScript applications, with a focus on mobile phones and tablets. It's built on top of Backbone.js.

Video Demo

Anagen is a sample application built using Tres, which is included in the source code.

Installation

Downloads are hosted on GitHub.

In the zip file, you'll find tres.js along with it's dependencies in the js/ folder. Make sure you include all of those in your application.

Base styles and the default theme can be found in the css/ folder also in the zip file. You don't need to use the default theme of course, though I highly recommend you include css/tres/base.css at least.

A packaging system as well as a development environment is underway. This will get considerably easier then.

How to use

The simplest Tres app looks like this:

class HomeScreen extends Tres.Screen
  template : _.template('<h1>Hello there</h1>')

$ ->
  App = new Tres.App
  App.on '' : new HomeScreen

  App.boot()

The workflow follows much of this recipe: subclass from Tres.Screen, declare a template which holds the markup for a screen, and optionally define what happens in a screen.

Tres provides a few classes for making common patterns easy to implement.

General

Q: How do I make a Tres app?

A: First, follow the instructions from the install section.

You can write a simple app following the recipe right at the start of the how to use section. The idea is this: 

class Home extends Tres.Screen
class About extends Tres.Screen

$ ->
  App = new Tres.App
  App.on
    ''        : new Home   # Home URL
    '/about'  : new About  # About page URL

The method App.on takes an object, which is used to map URLs to the screen objects your app has.

Tres.Screen

Q: How do I navigate to another screen?

A: add a regular <a> tag to it's URL. Tres screen objects will automatically bind links you add in a screen's template.

For instance:

class Home extends Tres.Screen
  template : _.template('<a href="about">Go to about<a/>')
class About extends Tres.Screen

$ ->
  App = new Tres.App
  App.on
    ''      : new Home
    'about' : new About

  App.boot()

Tapping the link on that page will have the app transition to the About screen.
